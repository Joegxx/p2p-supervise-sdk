# p2p-supervise-sdk
P2P网贷机构实时数据接入平台 （互联网应急中心）sdk retrofit封装

## 注意
下面pom的依赖。是官网的工具包。只是上传了自己的私服。请自行解决。

~~~
        <!-- 应急中心工具包-->
        <dependency>
            <groupId>org.cert</groupId>
            <artifactId>supervise</artifactId>
            <version>1.0.0</version>
            <type>jar</type>
        </dependency>
        
~~~

## 使用
  参见test代码。配置自家apikey和sourceCode。即可使用。
